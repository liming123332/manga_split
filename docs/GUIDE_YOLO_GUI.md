# GUI 使用指南 - YOLO & SAM 双模式

## 新功能概述 ✨

浮浮酱已经给 GUI 添加了 **YOLO/SAM 检测方法切换** 功能呢！

现在主人可以在界面上轻松选择检测方法喵～ ฅ'ω'ฅ

## 界面说明

### 1. 检测方法选择（新增）

位置：配置选项区域，"输入目录" 和 "输出目录" 下方

**两个选项：**
- ✅ **YOLO (快速推荐)** - 使用 YOLO 进行人脸检测
  - ⚡ 速度快（10-20ms/图）
  - 👍 精度良好
  - 💾 显存占用低

- ✅ **SAM (高精度)** - 使用 SAM 进行人物分割
  - 🐢 速度较慢（1-3s/图）
  - 🌟 精度优秀
  - 💾 显存占用高

### 2. YOLO 模型路径（选择 YOLO 时显示）

选择 YOLO 方法后，会显示：
- **YOLO 模型** - 输入框 + 浏览按钮
  - 默认：`./models/yolov8x6_animeface.pt`
  - 点击"浏览..."可选择其他模型文件

### 3. SAM 模型选择（选择 SAM 时显示）

选择 SAM 方法后，会显示：
- **SAM 模型** - 三个单选按钮
  - vit_h (高精度)
  - vit_l (平衡)
  - vit_b (快速)

## 使用流程

### 步骤 1：启动 GUI

```bash
python gui.py
```

### 步骤 2：选择输入/输出目录

1. 点击"输入目录"旁的"浏览..."按钮，选择漫画文件夹
2. 点击"输出目录"旁的"浏览..."按钮，选择输出文件夹

### 步骤 3：选择检测方法

**推荐设置（快速批量处理）：**
- ✅ 选择 **YOLO (快速推荐)**
- ✅ 确认 YOLO 模型路径正确
- ✅ 选择运行设备（CUDA 或 CPU）
- ✅ 点击"开始处理"

**高精度设置（重要图像）：**
- ✅ 选择 **SAM (高精度)**
- ✅ 选择 SAM 模型类型（推荐 vit_l）
- ✅ 选择运行设备
- ✅ 点击"开始处理"

### 步骤 4：监控进度

- 进度条显示处理进度
- 日志窗口显示详细信息
- 可随时点击"停止"按钮中断处理

## 高级设置

点击"高级设置..."按钮可以调整：

- **长条图分割**
  - 启用/禁用长条图分割
  - 分割参数

- **人物检测**
  - 人物最小尺寸
  - 单张最大检测人数

- **后处理**
  - 边缘羽化半径
  - 是否裁剪白边
  - 裁剪边距

## 实时切换

处理过程中可以实时切换检测方法喵～

**切换效果：**
- 选择 **YOLO** → 界面立即显示 YOLO 模型路径
- 选择 **SAM** → 界面立即显示 SAM 模型选项

## 配置保存

当前 GUI 的配置**不会自动保存**到文件。

如果想要持久化配置：
1. 手动编辑 `config.yaml`
2. 或者在 GUI 中每次重新选择

## 常见问题

### Q1: 切换检测方法后需要重启吗？

**A:** 不需要！可以随时切换，立即生效。

### Q2: YOLO 模型文件在哪里？

**A:** 默认位置：`./models/yolov8x6_animeface.pt`
- 如果文件不存在，点击"浏览..."选择已下载的模型
- 或使用标准模型 `yolov8n.pt`（会自动下载）

### Q3: 哪种方法检测精度更高？

**A:**
- **YOLO** - 专门针对二次元人脸训练，精度良好
- **SAM** - 通用分割模型，精度优秀，但速度慢

### Q4: 可以批量处理时先用 YOLO，失败图像再用 SAM 吗？

**A:** 目前 GUI 不支持自动切换。建议：
1. 全部用 YOLO 快速处理
2. 检查输出结果
3. 对检测失败的图像单独用 SAM 处理
